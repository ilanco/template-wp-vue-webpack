# template-wp-vue-webpack

> A modern starter theme for Wordpress which uses Vue 2, Vuex, Vue-router and Webpack 2

## Usage

Install [SAO](https://github.com/egoist/sao) first.

### From git

```bash
sao ilanco/template-wp-vue-webpack
```

### From npm

```bash
sao wp-vue-webpack
```
